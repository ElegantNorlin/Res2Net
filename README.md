# Res2Net
本仓库为Res2Net模块的代码，以学习为主，原GitHub仓库为https://github.com/yfreedomliTHU/Res2Net
在学习过程中对代码进行了详细的注释

与原仓库不同的是，我的代码是单纯的res2模块，并没有结合其他的CNN网络。

本仓库Res2Net的代码可以打印出每一层的特征维度参数
可以根据代码逻辑打印其他CNN网络中各层的特征维度参数，为网络调整带来了便携
